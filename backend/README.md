# GDGC Backend (JSON)

## Run locally
1. cd backend
2. npm install
3. npm run dev   # requires nodemon
   or
   npm start

API endpoints:
- GET /members
- GET /members/:id
- GET /health

Query examples:
- /members?q=shubham
- /members?role=Frontend Developer
- /members?skill=React
- /members?location=Indore
